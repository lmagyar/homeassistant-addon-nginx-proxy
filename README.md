## Installation

Navigate in your Home Assistant frontend to **Settings** -> **Add-ons** ->
**Add-on Store**, in the **...** menu at the top right corner click
**Repositories**, add
`https://github.com/lmagyar/homeassistant-addon-nginx-proxy` as repository.
